基于Darknet、Flask做的PCB瑕疵检测   

运行步骤:   
1.安装conda     
2.conda create -n pcb python=3.6     
3.conda activate pcb      
4.pip install -r requirements.txt     
5.python pcb.py    
6.python app.py    
7.网页打开http://127.0.0.1:42366（官方网站https://pcb.z2lab.com/）
